Welcome to the **Applied Speech Processing & Audio Analysis** repository! This comprehensive crash course and lab environment is designed to take you from the fundamentals of digital signal processing to building advanced machine learning models for audio, including Speech Emotion Recognition, Wake Word Detection, and Speech-to-Text systems.

## 🚀 Overview
Audio data is unstructured and complex. This repository provides a step-by-step curriculum, utilizing industry-standard libraries (like `librosa`, `scikit-learn`, and `numpy`), to extract meaningful features from sound waves and build predictive models. 

Each module contains an interactive **Jupyter Notebook** and a dedicated `slides/` folder for theoretical background.

## 🗺️ Course Roadmap & Curriculum

- **Phase 1: Foundations of Audio Processing**
  - 📂 `01-Dissecting-Sound`: Understanding sampling rates, bit depth, and basic audio I/O.
  - 📂 `02-FFT-Audio-Analysis`: Fast Fourier Transform (FFT) and frequency domain analysis.
  - 📂 `03-Spectrogram`: Visualizing audio signals over time and frequency.

- **Phase 2: Feature Extraction**
  - 📂 `04-MFCC-Feature-Extraction`: Extracting Mel-Frequency Cepstral Coefficients (MFCCs).
  - 📂 `05-ML-Features`: Preparing audio features (Chroma, Mel, Tonnetz) for Machine Learning pipelines.

- **Phase 3: Applied Machine Learning Models**
  - 📂 `06-Speech-Emotion-Recognition`: Classifying human emotion using the RAVDESS dataset.
  - 📂 `07-Voice-Lie-Detector`: Building a binary classifier for stress/deception detection in voice.
  - 📂 `08-Wake-Word-Detection`: Creating a lightweight trigger-word detection system.
  - 📂 `09-Speech-to-Text`: Implementing automated speech recognition (ASR) basics.

## ⚙️ Getting Started

### Prerequisites
Make sure you have Python 3.8+ installed. It is highly recommended to use a virtual environment.

### Installation
1. Clone the repository:
   ```bash
   git clone [https://github.com/your-username/Applied-Speech-Processing-in-Python.git](https://github.com/your-username/Applied-Speech-Processing-in-Python.git)
   cd Applied-Speech-Processing-in-Python

```

2. Install the required dependencies:
```bash
pip install -r requirements.txt

```


3. Launch Jupyter Notebook:
```bash
jupyter notebook

```
